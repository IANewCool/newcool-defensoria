# NewCool Defensoria - Defensoria Penal Publica

Portal educativo sobre la Defensoria Penal Publica de Chile.

## Demo

[https://defensoria.newcool.io](https://defensoria.newcool.io)

## Caracteristicas

- Verificador de requisitos DPP
- Guia de derechos del imputado
- Informacion del proceso penal
- Ubicacion de defensorias
- Consulta de casos

## Tecnologias

- Next.js 15
- React 19
- Tailwind CSS 4
- TypeScript

## Desarrollo

```bash
npm install
npm run dev
```

## Parte del Ecosistema NewCool

Este modulo es parte de [NewCooltura Informada](https://informada.newcool.io) - Portal de instituciones publicas de Chile.
